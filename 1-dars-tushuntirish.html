Lesson: HTML Form Elementlari

Formadagi asosiy teglar va ularning vazifasi

1. Form tagi

<form action="test.html" method="get"> ... </form>
Bu element formaning boshlanishi va tugashini bildiradi. action → ma’lumot qaysi faylga yuborilishini ko‘rsatadi.
method → ma’lumot yuborish usuli (GET yoki POST).

2. Matn kiritish

<input name="ism" type="text">

type="text" → foydalanuvchi matn kiritadi. name → serverga yuboriladigan ma’lumot nomi.

3. Telefon (raqam) kiritish

<input name="tel" type="number">

type="number" → faqat son kiritish imkonini beradi.

4. Radio tugmalar

Kompyuter Telefon 
<input type="radio" name="qurilma">

Radio tugmalar → bir xil name bo‘lsa, faqat bitta tanlanadi.

5. Checkbox tugmalar

<input type="checkbox">

Checkbox → bir nechta variantni tanlash mumkin.

6. Sana tanlash

<input type="date">

type="date" → foydalanuvchi taqvimdan sana tanlaydi.

7. Email kiritish

<input type="email">

type="email" → email formatini tekshiradi (test@example.com).

8. Vaqt tanlash

<input type="time">

type="time" → foydalanuvchi vaqt (soat:dakika) tanlaydi.

9. Tugmalar


<button type="submit">Ok</button>
<button type="reset">Clear</button>

submit → formani serverga yuboradi.
reset → barcha maydonlarni tozalaydi.
